# Open Source Maintainer's First Mate

A hackathon project built for the Pirates of the Coral-bean Hackathon.

## Features

- Fetches live GitHub issues
- Helps maintainers inspect repositories
- Built using Streamlit and Python
- AI-powered maintainer assistant (in progress)

## Tech Stack

- Python
- Streamlit
- GitHub API

## Run Locally

```bash
pip install -r requirements.txt
streamlit run app.py
